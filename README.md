# Develop Tools Pro (DTP)

一个现代化的开发者工具集合网站，采用 Bento Grid 设计风格，为开发者提供高效、便捷的在线工具体验。

## ✨ 特性

- 🎨 **现代化设计** - 采用 Bento Grid 布局，视觉清爽，交互流畅
- 🛠️ **30+ 实用工具** - 涵盖 JSON、时间、编码、文本处理等多个类别
- 📱 **响应式设计** - 完美适配桌面端和移动端
- ⚡ **纯前端实现** - 无需后端，所有功能在浏览器端完成
- 🔍 **智能搜索** - 支持工具名称、描述、分类的模糊搜索
- 🎯 **分类筛选** - 按工具类别快速筛选
- 🌍 **智能多语言** - 支持中文和英文界面切换，根据IP地址自动检测用户地区
- 📤 **一键分享** - 支持微信、QQ、微博等多种分享方式
- 💬 **意见反馈** - 用户反馈收集和管理系统
- 📊 **数据统计** - Google Analytics 集成，支持用户行为分析

## 🚀 快速开始

### 环境要求

- Node.js >= 16.0.0
- npm >= 8.0.0

### 安装依赖

```bash
npm install
```

### 启动开发服务器

```bash
npm run dev
```

访问 http://localhost:3000 查看项目

### 构建生产版本

```bash
npm run build
```

### 预览生产版本

```bash
npm run preview
```

## 🌟 核心功能

### 📤 一键分享功能

让用户可以轻松分享网站到各种社交媒体平台，提升网站的传播效果和用户参与度。

#### 分享方式
- **复制链接** - 一键复制网站链接到剪贴板
- **微信分享** - 通过微信分享链接
- **QQ分享** - 通过QQ分享链接
- **微博分享** - 通过微博分享链接
- **朋友圈分享** - 提供朋友圈分享提示
- **二维码分享** - 生成网站二维码供扫描

#### 功能特色
- 🎯 **智能统计** - 记录用户分享的总次数
- 💾 **本地存储** - 使用 localStorage 持久化分享数据
- 🎨 **美观界面** - 绿色主题悬浮按钮，与反馈按钮对称布局
- 📱 **响应式设计** - 完美适配桌面和移动端
- 🌍 **多语言支持** - 支持中英文界面切换

### 💬 意见反馈系统

完整的用户反馈收集和管理系统，帮助开发者了解用户需求和改进产品。

#### 用户功能
- **悬浮反馈按钮** - 右下角紫色反馈按钮，随时可访问
- **反馈类型选择** - 支持建议、问题、功能、其他四种类型
- **评分系统** - 1-5星评分，量化用户满意度
- **详细描述** - 支持最多1000字符的详细反馈内容
- **实时提交** - 反馈内容实时保存到本地存储

#### 管理功能
- **反馈管理页面** - 访问 `/admin/feedback` 查看所有反馈
- **统计面板** - 显示总反馈数、待处理、已查看、已解决数量
- **筛选功能** - 按状态和类型筛选反馈
- **状态管理** - 标记为已查看、已解决
- **数据导出** - 支持导出反馈数据
- **批量操作** - 支持批量标记和删除

#### 技术特性
- 🔒 **隐私保护** - 所有数据存储在用户本地，保护隐私
- 📊 **数据统计** - 平均评分、反馈分布等统计信息
- 🎨 **现代UI** - 毛玻璃效果、流畅动画
- 🌍 **多语言** - 完整的中英文支持

### 🌍 智能语言检测

根据用户IP地址自动检测所在地区，智能推荐最适合的语言界面。

#### 检测逻辑
- **中国大陆** - 自动切换到中文界面
- **台湾、香港、澳门** - 自动切换到中文界面
- **新加坡、马来西亚** - 自动切换到中文界面（华人较多）
- **其他地区** - 默认使用英文界面

#### 功能特色
- **自动检测** - 首次访问时自动检测用户地区
- **智能缓存** - 24小时缓存地理位置信息，避免重复请求
- **用户偏好** - 尊重用户手动设置的语言偏好
- **多API备选** - 使用多个IP检测服务确保可靠性
- **隐私保护** - 不存储用户IP地址，仅用于语言推荐

#### 技术实现
- **IP检测服务** - ipify.org、ipapi.co、ipinfo.io
- **地理位置API** - 多个备选API确保服务可用性
- **本地存储** - 缓存检测结果和用户偏好
- **降级处理** - 检测失败时使用默认语言

- **前端框架**: Vue 3 + Composition API

- **CSS框架**: Tailwind CSS
- **图标库**: Lucide Vue
- **国际化**: Vue I18n
- **路由**: Vue Router
- **工具库**: 
  - dayjs - 日期时间处理
  - js-yaml - YAML解析
  - highlight.js - 代码高亮
  - crypto-js - 加密解密
  - qrcode - 二维码生成
- **分析工具**: Google Analytics 4
- **部署平台**: Vercel

## 📝 开发指南

### 添加新工具

1. 在 `src/data/tools.js` 中添加工具配置
2. 在 `src/components/tools/` 中创建工具组件
3. 在 `src/views/ToolDetail.vue` 中注册组件映射

### 工具组件规范

```vue
<template>
  <div class="space-y-6">
    <!-- 工具内容 -->
  </div>
</template>

<script setup>
// 工具逻辑
</script>
```

### 添加新功能

#### 多语言支持
1. 在 `src/locales/zh-CN.js` 和 `src/locales/en.js` 中添加翻译
2. 在组件中使用 `useI18n()` 获取翻译函数
3. 使用 `t('key')` 显示翻译文本

#### 反馈系统扩展
1. 修改 `src/utils/feedbackStorage.js` 扩展数据结构
2. 更新 `src/components/FeedbackButton.vue` 添加新字段
3. 在 `src/views/FeedbackAdmin.vue` 中添加管理功能

#### 分享功能扩展
1. 在 `src/components/ShareButton.vue` 中添加新的分享平台
2. 更新分享链接格式和参数
3. 添加相应的多语言翻译

### 部署配置

#### Vercel 部署
- 项目已配置 `vercel.json` 支持 SPA 路由
- 静态文件（favicon、ads.txt）已配置正确的路由规则
- 支持环境变量配置

#### 环境变量
```bash
# Google Analytics
VITE_GA_MEASUREMENT_ID=G-LC13BPKH3G

# Google AdSense
VITE_ADSENSE_CLIENT_ID=ca-pub-3896009904827809
```
