工具概述

API接口Mock生成器是一个强大的开发工具，旨在帮助开发者快速生成逼真的API接口Mock数据。该工具集成了@faker-js/faker库，支持多种数据类型，并提供网络延迟模拟功能，让开发者能够更真实地测试API接口。

使用指南

  1. 配置API端点：输入API端点路径，选择HTTP方法（GET、POST、PUT等）
  2. 定义字段结构：点击"添加字段"按钮，为每个字段设置名称、类型和描述
  3. 设置网络延迟：使用滑块调整网络延迟时间，或选择预设延迟值
  4. 生成Mock数据：点击"生成Mock数据"按钮，等待延迟时间后查看结果
  5. 查看结果：在选项卡中查看JSON数据、cURL命令或JavaScript代码
  6. 复制使用：点击相应的复制按钮，将代码复制到剪贴板使用

使用技巧

选择合适的字段类型：根据实际API需求选择对应的数据类型，确保生成的数据符合预期格式。
网络延迟设置：根据目标用户网络环境设置合适的延迟时间，测试不同网络条件下的用户体验。
批量字段操作：使用"加载示例"功能快速创建常见的API结构，然后根据需要进行调整。
代码生成：生成的cURL和JavaScript代码可以直接用于API测试，提高开发效率。
数据一致性：每次生成的数据都是随机的，但格式保持一致，适合测试各种数据场景。
注意事项：生成的数据仅用于开发和测试，请勿在生产环境中使用真实用户数据。

技术实现

核心技术栈

  • Vue 3 Composition API：现代化响应式开发框架
  • @faker-js/faker：强大的假数据生成库
  • 纯前端实现：无需后端服务，完全在浏览器中运行
  • 响应式设计：支持桌面端和移动端
  • 多语言支持：完整的中英文界面
