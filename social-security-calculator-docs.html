使用指南

  1. 选择城市：从下拉菜单中选择您所在的城市，系统会自动加载该城市的社保公积金比例数据。
  2. 输入工资：在工资输入框中输入您的月薪金额（税前），支持整数和小数输入。
  3. 查看结果：系统会自动计算并显示各项明细，包括个人缴纳、公司缴纳、税后收入和公司总支出。
  4. 分析数据：通过总览卡片了解关键数据，通过详细表格了解具体缴纳项目。
  5. 参考比例：查看进度条了解各项占工资的比例，帮助您更好地理解薪酬结构。
  6. 注意事项：计算结果仅供参考，具体以当地政策为准。

使用技巧

城市选择：不同城市的社保公积金比例不同，请选择正确的城市以获得准确的计算结果。
数据准确性：计算结果仅供参考，具体以当地政策为准。实际缴纳比例可能因政策调整而变化。
政策更新：建议定期查看当地社保局和公积金管理中心的最新政策，确保数据的时效性。
薪酬结构：个人缴纳部分会从工资中扣除，影响税后收入。公司总支出包括工资和公司缴纳的社保公积金部分。
住房公积金：住房公积金比例在不同城市差异较大，是影响总支出的重要因素。
保险覆盖：工伤保险和生育保险通常由公司全额承担，个人无需缴纳。
