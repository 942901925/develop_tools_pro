🎯 假期计算器 - 新计算逻辑优化

📊 计算逻辑对比
❌ 旧逻辑（固定长度）

• 固定使用所有可用假期天数

• 不考虑周末和节假日的搭配

• 可能错过更好的优化方案

• 计算方式：总天数 = 假期天数

✅ 新逻辑（智能搭配）

• 休假天数可以搭配周末和节假日

• 自动寻找最优的连续日期段

• 最大化实际休息天数

• 计算方式：总天数 = 假期天数 + 周末 + 节假日

🚀 新功能特性
智能优化算法：
  • 遍历所有可能的连续日期段（从可用天数到2倍长度）
  • 自动识别周末和节假日
  • 选择总休息天数最多的方案
  • 确保实际使用的假期天数不超过可用天数
评级系统优化：
  • A+：节省天数 ≥ 使用天数（100%+）
  • A：节省天数 ≥ 70%使用天数
  • B+：节省天数 ≥ 50%使用天数
  • B：节省天数 ≥ 30%使用天数
  • C+：节省天数 ≥ 10%使用天数
  • C：节省天数 < 10%使用天数
用户体验提升：
  • 添加计算说明，让用户理解新逻辑
  • 方案对比表格增加节省天数列
  • 更直观的数据展示
  • 支持多方案保存和对比
